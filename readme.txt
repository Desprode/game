绘制背景，绘制海葵，海葵的位置随机，高度随机，但是有一定的范围，使用贝塞尔曲线使海葵摆动，绘制食物，一共30个食物，15个活动的，15个不活动的，当大鱼吃掉一个食物时，便选择一个食物激活，画布始终保持15个活动着的食物，绘制大鱼，大鱼是由好多张图片组成的，使用定时器来切换大鱼的颜色，大鱼的眼睛，使用外部函数库使大鱼跟随鼠标移动时更加自然，当大鱼碰到食物时，使用外部函数库的碰撞检测，检测到碰撞时在食物的位置，绘制一个特效的圆，半径逐渐增大，增大到一定程度时消失，小鱼的绘制于大鱼类似
项目名称：大鱼和小鱼
项目时间：10天左右
项目描述：
    1.使用canvas绘图技术，将游戏中静态的对象绘制出来
    2.结合js，使游戏中能够流畅的运行
项目遇到的问题：
    1.如果只使用canvas中的技术，海葵摆动不自然
        解决：使用贝塞尔曲线，使海葵摆动更加自然
    2.大鱼在跟随鼠标游动时，显得僵硬，而且不会转弯
        解决：引入外部函数库
项目收获：
    1.熟悉了canvas的使用
    2.学习到一些新的知识，如贝塞尔曲线
